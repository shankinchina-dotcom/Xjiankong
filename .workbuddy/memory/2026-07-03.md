## 关键技术发现

### TrendRadar frequency_words.txt 解析逻辑
- `[GLOBAL_FILTER]` 和 `[WORD_GROUPS]` 是 section marker，`current_section` 变量控制当前区域
- 缺 `[WORD_GROUPS]` 时，`current_section` 停留在 `GLOBAL_FILTER`，后续所有内容（包括 AI 关键词）被当作全局过滤词
- `[组别名]`（如 `[AI 核心]`）不是 section marker，只是词组显示名
- 正则语法 `/\bAI\b/i` 完整支持，`re.compile(pattern, re.IGNORECASE).search()` 实现
- `+正则` 是必须词，`@N` 是该组最多显示 N 条

### 容器调试技巧
- `docker exec trendradar python3 -c "..."` 可直接在容器内跑 Python 验证解析结果
- 容器以只读方式挂载 Xjiankong config 目录，改源文件后需 `docker restart trendradar`

   - 每日/每周/每月自动更新，源头 huggingface.co/papers
